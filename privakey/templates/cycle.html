The Vigenere cipher is similar to the classic Caesarian shift but allows for multiple values of offset. The image below shows what each shift would look like.

For example, a cycle of length 3 allows 3 different offsets. If one were to choose 1, 2, and 3 as their offsets, then the first letter would be replaced with the next letter in the alphabet. Then, the second letter would be replaced with the letter that is two positions higher in the alphabet, the third letter would be replaced with the letter that is 3 positions higher and the cycle repeats. Using this cycle and these shifts, the word "cereal" would then become "dgufco".
