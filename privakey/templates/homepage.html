Ciphers

Statement

Dadboyz created this encryption web app to allow for more privacy between users. With some ciphers, all that you need is the keyword and the message. Then, simply go to the correct page, input your encrypted message and the code word, and receive the original message.

History

Ciphers have been used all through out history. They have helped groups keep sensitive information safe from others by making a message readable only to a select few. Here, we have replicated a few types of ciphers for you.

Details

On the left sidebar, you will see a list of the ciphers we have to offer. On each's individual page, there is a description on how it works followed by the appropriate boxes to customize it the way you want. Then, you simply enter the message you want to encrypt or decrypt and recieve the new message.
